## Build & Run

1. Build
```sh
git clone https://github.com/qubus-ai/qubus.git
npm install
npm run build-release
```
2.a.  Run
```sh
npm run qubus
```
2.b. Build and run in production mode
```sh
npm run build-prod
```

2.b. Build and run in developlment mode (opened chrome devTools)
```sh
npm run build-dev
```

2.b. Build and run in developlment mode (opened chrome devTools)
```sh
npm run build-dev
```

2.c. Build and run with front-end served by webpack-dev-server (changes automatically reloaded)
```sh
npm run build-dev
npm run qubus-remote
```
